Setting Up Chat

Download & Install

First, ensure your device is connected to the Why-Fi and you have download the Element Messenger app.

If you don't have the app, pay more attention to the announcement channel next time ü§ó, you won't be able to download it from the desert

How to use

  • Open the Element App
  • Choose create an account. When asked about the type of use, you can skip this question
  • On the next screen, under "where you're conversations will live, put 
    http://whynotpi.local:8008
  • Choose your name as user name (so people can text you), any password and email. You won't have to confirm your email
  • Optionally, personalize your profile
  • When at the welcome screen, you initiate new private chats or join chat rooms (like general) from the new message / note icon in the top bottom üìù
  • Click explore rooms to join the general chat
  • Click start chat to invite someone to chat privately
  • Click start create room if you want to set up a new chat for a specific topic
  • Please use the media sharing (images, videos) functionality sparingly. It is not the place to exchange our party pictures

If you had previously created an account (before going to the desert)

  • You'll need to log out and start from scratch as your registered with a different server
